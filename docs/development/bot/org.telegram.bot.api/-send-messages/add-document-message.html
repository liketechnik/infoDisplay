addDocumentMessage

open fun addDocumentMessage(messageHash: Int, caption: String, chatId: String, absSender: AbsSender, fileId: String, inlineKeyboard: Optional<InlineKeyboardMarkup>): Unit (source)

Adds a new SendDocument to the queue of messages that are going to be send.

Parameters

messageHash - A unique hash of the message text. Used to identify which parameter belongs to which message.

caption - The caption shown below the document.

chatId - The id of the chat the message is send to.

absSender - The AbsSender with which the message is send.

fileId - The Telegram fileID of the document that is send.

inlineKeyboard - (Optional) An InlineKeyboardMarkup appended to the message.

calculateHash

open fun calculateHash(): Int (source)

Calculate a unique hash from the text of the message. Uniqueness is guaranteed by synchronizing this method to Message.class, adding the System#currentTimeMillis() to the message text before calculating the hash and waiting one millisecond before the hash is returned.

Exceptions

InterruptedException - If sleeping before returning the message is interrupted.

Return
The unique hash of the message text.

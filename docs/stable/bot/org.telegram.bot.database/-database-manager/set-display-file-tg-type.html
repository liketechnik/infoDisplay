setDisplayFileTgType

fun setDisplayFileTgType(displayFileName: String, tgType: String): Unit (source)

Set the type of message the media was sent over Telegram. Possible types are Config.Bot#DISPLAY_FILE_TG_TYPE_AS_DOCUMENT if it was sent as a document, Config.Bot#DISPLAY_FILE_TG_TYPE_IMAGE if it was sent as an image or Config.Bot#DISPLAY_FILE_TG_TYPE_VIDEO if it was sent as a video.

Parameters

displayFileName - The title of the display file.

tgType - The type of the message that contained the media saved as specified display file.

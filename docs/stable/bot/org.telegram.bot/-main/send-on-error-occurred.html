sendOnErrorOccurred

open static fun sendOnErrorOccurred(absSender: AbsSender, user: User, chat: Chat, LOGTAG: String): Unit (source)

This method is called when an error occurs in one of the bot commands. It tells the user about the occurrence of an error and prints out the help message.

Parameters

absSender - Needed to send a message to the user.

user - The user the message should go to.

chat - The chat the message should be send to.

LOGTAG - The LOGTAG of the command the error occurred in.

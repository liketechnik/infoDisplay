open fun execute(absSender: AbsSender, user: User, chat: Chat, arguments: Array<String>): Unit (source)

Send the user a message with a media file and a keyboard with buttons for selecting different media files or deleting the current one. The media file that is send is determined either by evaluating the callback data passes by org.telegram.bot.CallbackParser or set to index 0 when no data is available.

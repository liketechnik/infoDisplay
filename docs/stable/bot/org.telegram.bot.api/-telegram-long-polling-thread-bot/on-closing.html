onClosing

open fun onClosing(): Unit (source)

Called when the bot session is closed. Logs that the shutdown process is initiated, waits some time to let the commands shut down. Then lets the SendMessages thread do its work. During the whole process it logs the current status, so the user knows how long it will take till the bot shuts down completely.

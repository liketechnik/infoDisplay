addEditMessage

open fun addEditMessage(messageHash: Int, message: String, chatId: String, absSender: AbsSender, messageID: Int, enableMarkdown: Optional<Boolean>, inlineKeyboard: Optional<InlineKeyboardMarkup>): Unit (source)

Adds a new EditMessageText to the queue of messages that are going to be send.

Parameters

messageHash - A unique hash of the message text. Used to identify which parameter belongs to which message.

message - The new message text of the message.

chatId - The id of the chat the message is in.

absSender - The AbsSender with which the messages is edited.

messageID - The id of the message that is changed.

enableMarkdown - (Optional) If the message text should be treated as markdown formatted text.

inlineKeyboard - (Optional) An InlineKeyboardMarkup appended to the message.
